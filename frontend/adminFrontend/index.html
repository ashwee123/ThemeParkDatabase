Export (CSV)

CSV buttons download row-level data. The PDF button uses the same snapshot and thresholds as the grid above.

  • Visitors — guest directory (IDs, contact, active flag). Activity count columns download as 0 in this export so large databases do not time out.
  • Tickets — issued tickets with visitor, type, price, dates.
  • Incidents — staff incident reports for safety / ops review.
  • Visitor reviews — area ratings and comments for HR and quality.
